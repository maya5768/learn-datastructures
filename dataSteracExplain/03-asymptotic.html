מה הם חסמים אסימפטוטיים?

כאשר אנו מנתחים אלגוריתמים, אנחנו רוצים לדעת איך זמן הריצה גדל ככל שגודל הקלט גדל. החסמים האסימפטוטיים מאפשרים לנו לחסום את הגידול של פונקציה מכמה כיוונים: מלמעלה, מלמטה, או משניהם.

שלושת החסמים העיקריים: O, Θ, Ω.

חסם עליון – Big O

הגדרה פורמלית:

עבור שתי פונקציות f(n) ו-g(n), נאמר ש-f(n) = O(g(n)) אם קיימים קבועים חיוביים c ו-n₀ כך שלכל n ≥ n₀ מתקיים:

f(n) ≤ c · g(n)

במילים אחרות, g(n) "חוסמת" את f(n) מלמעלה עבור ערכים גדולים מספיק של n.

דוגמה:

האם f(n) = 2n + 3 = O(n²)?

צריך למצוא קבועים c ו-n₀ שיקיימו: 2n + 3 ≤ cn²

נבחר c = 2. האם קיים n₀ שממנו מתקיים 2n + 3 ≤ 2n²?
כן, נבחר למשל n₀ = 5.

✅ לכן 2n + 3 = O(n²)

חסם תחתון – Big Ω

הגדרה פורמלית:

עבור שתי פונקציות f(n) ו-g(n), נאמר ש-f(n) = Ω(g(n)) אם קיימים קבועים חיוביים c ו-n₀ כך שלכל n ≥ n₀ מתקיים:

0 ≤ c · g(n) ≤ f(n)

כלומר, f(n) אינה קטנה מ-g(n) כפול קבוע, עבור n גדול מספיק.

דוגמה:

האם f(n) = 5n = Ω(10n)?

צריך למצוא c ש-c · 10n ≤ 5n, כלומר 2c ≤ 1.

נבחר c = 0.5 ואי השוויון יתקיים לכל n.

✅ לכן 5n = Ω(10n)

חסם הדוק – Big Θ

הגדרה פורמלית:

עבור שתי פונקציות f(n) ו-g(n), נאמר ש-f(n) = Θ(g(n)) אם קיימים שלושה קבועים חיוביים c₁, c₂, n₀ כך שלכל n ≥ n₀ מתקיים:

0 ≤ c₁ · g(n) ≤ f(n) ≤ c₂ · g(n)

כלומר, g(n) היא חסם הדוק עבור f(n) – גם מלמעלה וגם מלמטה.

דוגמה:

הוכיחו ש-5n + 3 = Θ(n)

נצטרך למצוא שני קבועים שמקיימים:

c₁ · n ≤ 5n + 3 ≤ c₂ · n

אפשר לבחור c₁ = 5, c₂ = 8, n₀ = 1, וכל אי השוויונות מתקיימים.

כללי חיבור וכפל חסמים

חיבור חסמים:

O(f(n)) + O(g(n)) = O(max(f(n), g(n)))
Ω(f(n)) + Ω(g(n)) = Ω(max(f(n), g(n)))
Θ(f(n)) + Θ(g(n)) = Θ(max(f(n), g(n)))

כפל חסמים:

O(f(n)) · O(g(n)) = O(f(n) · g(n))
Ω(f(n)) · Ω(g(n)) = Ω(f(n) · g(n))
Θ(f(n)) · Θ(g(n)) = Θ(f(n) · g(n))

משפט האב (Master Theorem)

משפט האב הוא כלי חזק לפתרון נוסחאות נסיגה מהצורה:

T(n) = a · T(n/b) + f(n)

כאשר a ≥ 1, b > 1, ו-f(n) היא פונקציה חיובית.

שלושת המקרים:

מקרה 1: f(n) קטנה מ-n^log_b(a)

אם קיים ε > 0 כך ש-f(n) = O(n^(log_b(a) - ε)), אזי:

T(n) = Θ(n^log_b(a))

מקרה 2: f(n) שווה ל-n^log_b(a)

אם f(n) = Θ(n^log_b(a)), אזי:

T(n) = Θ(n^log_b(a) · log n)

מקרה 3: f(n) גדולה מ-n^log_b(a)

אם קיים ε > 0 כך ש-f(n) = Ω(n^(log_b(a) + ε)), אזי:

T(n) = Θ(f(n))

דוגמאות לשימוש במשפט האב

דוגמה 1: T(n) = 8T(n/2) + n²

  • a = 8, b = 2, f(n) = n²
  • log_b(a) = log_2(8) = 3
  • האם f(n) = O(n^(3-ε))? נבחר ε = 1: f(n) = O(n²) ✅
  • מקרה 1 מתקיים, לכן: T(n) = Θ(n³)

דוגמה 2: T(n) = 2T(n/2) + n (Merge Sort!)

  • a = 2, b = 2, f(n) = n
  • log_b(a) = log_2(2) = 1
  • האם f(n) = Θ(n^1) = Θ(n)? כן ✅
  • מקרה 2 מתקיים, לכן: T(n) = Θ(n log n)

דוגמה 3: T(n) = 2T(n/2) + n²

  • a = 2, b = 2, f(n) = n²
  • log_b(a) = log_2(2) = 1
  • האם f(n) = Ω(n^(1+ε))? נבחר ε = 1: f(n) = Ω(n²) ✅
  • מקרה 3 מתקיים, לכן: T(n) = Θ(n²)

הוכחות נוספות חשובות

טרנזיטיביות של O:

אם f(n) = O(g(n)) וגם g(n) = O(h(n)) אז f(n) = O(h(n)).

הוכחה: מהנתון קיימים c₁, c₂ כך:

f(n) ≤ c₁·g(n) ≤ c₁·c₂·h(n)

לכן קיים c₃ = c₁·c₂ שמקיים את אי השוויון.

תכונת הלוג:

אם f(n) = O(g(n)) אז log(f(n)) = O(log(g(n))).

דוגמה להפרכה:

האם נכון שאם f(n) = O(g(n)) אז g(n) = O(f(n))?

לא נכון! דוגמה נגדית: f(n) = n, g(n) = n².
n = O(n²) (נכון), אך n² ≠ O(n).
