מה זו רקורסיה?

רקורסיה היא טכניקת תכנות שבה פונקציה קוראת לעצמה. כדי לתכנן רקורסיה נכונה, עלינו לחשב את נוסחת הנסיגה של הבעיה – כלומר, איך אפשר להביע את הפתרון לבעיה בגודל n באמצעות פתרון של בעיות קטנות יותר.

🔑 שני מרכיבי הרקורסיה

  • תנאי עצירה (Base Case) – מקרה בסיסי שאנחנו יודעים לפתור ישירות.
  • קריאה רקורסיבית – פתרון הבעיה באמצעות קריאה לעצמה עם קלט קטן יותר.

דוגמה 1: חישוב עצרת (Factorial)

נרצה לכתוב שיטה שמקבלת מספר שלם ומחזירה את העצרת שלו:

נוסחת הנסיגה:

T(1) = 1                    // תנאי עצירה
T(n) = T(n-1) * n          // קריאה רקורסיבית

פירוט:

T1 = 1! = 1
T2 = 2! = 1*2 = T1 * 2
T3 = 3! = 1*2*3 = T2 * 3
T4 = 4! = 1*2*3*4 = T3 * 4
...
Tn = n! = T(n-1) * n

מימוש ב-Java:

public static int T(int n) {
    if(n == 1) {          // תנאי עצירה
        return 1;
    }
    return T(n - 1) * n;   // קריאה רקורסיבית
}

ניתוח זמן ריצה:

זמן הריצה של פונקציה זו הוא O(n) – יש n קריאות רקורסיביות, ובכל קריאה מבוצעת פעולה אחת (כפל).

דוגמה 2: סכום איברי מערך

public int sumArray(int[] a, int i) {
    if(i == a.length)
        return 0;
    return a[i] + sumArray(a, i + 1);
}

נוסחת הנסיגה:

T(n) = T(n-1) + 1
T(0) = 1

פתרון:

בכל שלב הבעיה קטנה ב-1, ובכל שלב משקיעים O(1) עבודה. לכן סך הכל יש n שלבים של עבודה קבועה:

T(n) = T(n-1) + 1 = T(n-2) + 1 + 1 = ... = n = O(n)

עץ רקורסיה (Recursion Tree)

עץ רקורסיה הוא דרך ויזואלית לתאר את נוסחת הנסיגה. באמצעות העץ אפשר לקבל תמונה טובה יותר לגבי החישובים שצריך לבצע בפתרון.

הרעיון: כל פונקציה רקורסיבית היא צומת בעץ, וכל קריאה רקורסיבית היא התפצלות מהצומת.

דוגמה 3: פונקציה עם חלוקה לחצי

public int f(int[] a, int start, int end) {
    if(start == end)
        return a[start];
    int left = f(a, start, (start + end) / 2);
    int right = f(a, 1 + (start + end) / 2, end);
    return left + right;
}

נוסחת הנסיגה:

T(n) = 2T(n/2) + 1

פתרון באמצעות עץ רקורסיה:

רמה 0: T(n) – עבודה של 1
רמה 1: 2 × T(n/2) – עבודה של 2
רמה 2: 4 × T(n/4) – עבודה של 4
רמה k: 2^k × T(n/2^k) – עבודה של 2^k
גובה העץ: log n

סכום העבודה בכל הרמות:

T(n) = 1 + 2 + 4 + ... + 2^(log n) = 2n - 1 = O(n)

דוגמה 4: נוסחה מורכבת יותר

נפתור את הנוסחה:

T(n) = T(n/3) + T(2n/3) + n

בניית עץ רקורסיה:

רמה 0: n
רמה 1: n/3 + 2n/3 = n
רמה 2: n/9 + 2n/9 + 2n/9 + 4n/9 = n
...

בכל רמה יש עבודה של O(n)!

חישוב גובה העץ:

הגובה נקבע לפי הענף הארוך ביותר – במקרה זה, הענף שכל פעם מחלק ב-2/3:

גובה = log_(3/2)(n) = O(log n)

סיכום:

T(n) = O(n) * O(log n) = O(n log n)

💡 טיפים לפתרון נוסחאות נסיגה

  1. זיהוי המבנה: כתבו את נוסחת הנסיגה במפורש.
  2. ציור העץ: שרטטו את העץ עם כמה רמות ראשונות.
  3. חישוב עבודה בכל רמה: סכום העבודה בכל רמה.
  4. גובה העץ: חשבו כמה רמות יש בעץ.
  5. סכימה: סכמו את כל העבודות של כל הרמות.
