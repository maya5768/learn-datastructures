מה זו מחסנית?

מחסנית (Stack) היא מבנה נתונים שמאחסן אוסף של איברים תוך שמירה על סדר הכנסת האיברים. הגישה לאיברים מתבצעת בשיטת "הכנס אחרון – הוצא ראשון" (LIFO - Last In, First Out).

אפשר לדמיין מחסנית כערימת צלחות: מוסיפים צלחת למעלה ומוציאים צלחת מלמעלה. לא ניתן להגיע לצלחת בתחתית הערימה בלי להוציא קודם את כל הצלחות שמעליה.

הפעולות הבסיסיות

כל הפעולות על המחסנית מתבצעות בזמן O(1):

פעולה חתימה תיאור
push void push(int x) מוסיף את x לראש המחסנית
pop int pop() מוציא ומחזיר את האיבר שבראש המחסנית
isEmpty boolean isEmpty() מחזיר true אם המחסנית ריקה

⚠️ אם מנסים לבצע pop על מחסנית ריקה – יש לטפל בזה, למשל על ידי זריקת חריגה (Exception).

מימוש מחסנית באמצעות מערך

נממש מחסנית שמוגבלת בגודלה, עם כל הפעולות ב-O(1). נשתמש במערך בגודל maxSize ובמשתנה top שיציין את מיקום האיבר האחרון.

מימוש ב-Java:

public class Stack {
    private int maxSize;
    private int[] stackArray;
    private int top;
    
    // בנאי
    public Stack(int maxSize) {
        this.maxSize = maxSize;
        stackArray = new int[this.maxSize];
        top = -1;            // מחסנית ריקה
    }
    
    // בדיקה אם המחסנית מלאה
    public Boolean isFull() {
        return (top == maxSize - 1);
    }
    
    // בדיקה אם המחסנית ריקה
    public Boolean isEmpty() {
        return (top == -1);
    }
    
    // הוספת איבר
    public void push(int value) {
        if(isFull()) {
            System.out.print("המחסנית מלאה ולא ניתן להוסיף איבר");
        } else {
            stackArray[++top] = value;
        }
    }
    
    // הוצאת איבר
    public int pop() {
        if(isEmpty()) {
            System.out.print("המחסנית ריקה ולא ניתן להסיר איבר");
            return -1;
        } else {
            return stackArray[top--];
        }
    }
}

הסבר הקוד:

  • top = -1 בהתחלה מציין שהמחסנית ריקה.
  • top == maxSize - 1 → המחסנית מלאה.
  • stackArray[++top] = value → קודם מגדילים את top, אז שמים את הערך במקום החדש.
  • stackArray[top--] → מחזירים את הערך במיקום top, ואז מקטינים את top.

תרגיל: מציאת מקסימום במחסנית

שאלה: כתבו שיטה שמקבלת כפרמטר מחסנית s ומחזירה את האיבר המקסימלי בה. בסיום השיטה המחסנית חייבת להיות במצבה המקורי.

פתרון – שימוש במחסנית עזר:

public int findMaxInStack(Stack s) {
    Stack temp = new Stack();
    int max = Integer.MIN_VALUE;
    
    // מוציאים איברים מ-s ושומרים ב-temp תוך חיפוש מקסימום
    while(!s.isEmpty()) {
        int x = s.pop();
        if(x > max)
            max = x;
        temp.push(x);
    }
    
    // מחזירים את האיברים מ-temp בחזרה ל-s
    while(!temp.isEmpty())
        s.push(temp.pop());
    
    return max;
}

הסבר הקוד:

  1. מוציאים כל איבר מ-s, משווים ל-max, ושומרים ב-temp.
  2. בסוף הלולאה הראשונה, s ריקה ו-temp מכילה את הכל בסדר הפוך.
  3. מחזירים את כל האיברים מ-temp ל-s – כך הסדר חוזר למקור!

שימוש מעשי: חישוב ביטויים אלגבריים

מחסניות שימושיות מאוד לחישוב ביטויים עם סוגריים. נדגים על הביטוי:

( 2 + ( ( 5 + 3 ) * ( 2 * 4 ) ) )

האלגוריתם:

Stack vals = new Stack();    // מחסנית ערכים
Stack ops = new Stack();     // מחסנית אופרטורים

while(s != null) {
    if(isOperator(s))
        ops.push(s);           // אופרטור → מחסנית אופרטורים
    else if(isNumber(s))
        vals.push(s);          // מספר → מחסנית ערכים
    else if(s == ")") {      // סוגר סוגריים - מחשבים!
        String op = ops.pop();
        int v1 = vals.pop();
        int v2 = vals.pop();
        int res = perform(op, v1, v2);
        vals.push(res);        // תוצאה חזרה למחסנית
    }
    s = nextToken();
}

הסבר הקוד:

  • סוגר פותח ( – מתעלמים ממנו.
  • מספר – דוחפים למחסנית הערכים.
  • אופרטור (+, -, *, /) – דוחפים למחסנית האופרטורים.
  • סוגר סוגריים ) – מבצעים את הפעולה: מוציאים אופרטור, שני ערכים, מחשבים, ודוחפים חזרה.

✅ בסוף החישוב: (5+3) * (2*4) + 2 = 8 * 8 + 2 = 66 + 2 = 72

שימושים נפוצים במחסנית

  • בקשות חזרה (Undo/Redo) בתוכנות עריכה.
  • מימוש קריאות רקורסיביות – המחשב משתמש במחסנית.
  • בדיקת איזון סוגריים בטקסט/קוד.
  • חישוב ביטויים (Postfix, Infix).
  • היסטוריית דפדוף בדפדפן אינטרנט.
  • סריקת עצים – DFS בגרפים ועצים.
