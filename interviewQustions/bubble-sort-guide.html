למה O(n²)?
לולאה בתוך לולאה — הלולאה החיצונית רצה n פעמים, והפנימית n פעמים גם. n × n = n².
swap זהה!
זו בדיוק אותה פונקציית swap מ-Reverse Array — זו הסיבה שהפרדת אחריות שווה: כותבים פעם אחת ומשתמשים בכל מקום.
סיבוכיות זמן — Bubble Sort

שתי לולאות מקוננות — החיצונית רצה n פעמים, הפנימית n פעמים בממוצע. המכפלה היא n², לכן O(n²) במקרה הגרוע.

שלושת המקרים
מקרה טובO(n)

המערך כבר ממוין — מעבר אחד בלי החלפות, ה-swapped flag מפסיק הכל.

מקרה ממוצעO(n²)

מערך אקראי — בממוצע n²/2 השוואות, שמפשט ל-O(n²).

מקרה גרועO(n²)

המערך ממוין הפוך — כל זוג דורש החלפה בכל מעבר.
